## Aperçu

### Démarrer

1. Téléchargez le dossier depuis le repo
2. Ouvrez le dossier dans cmd tapez la commande npm i
3. Puis npm star pour l'exécuter

NB : assurez-vous que NodeJs et NPM sont bien installés sur votre machine.
